## Dependencies

- Next.js
- React
- Typescript
- Tailwindcss
- EsLint
- Prettier
- React Icons
- Framer Motion
- Yet Another React Lightbox

## Run Locally

Clone the project and go to the project directory

```bash
  cd damac-frontend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
